Перед вами - последняя созданная версия лингвистического бота Discord!
Целью этого бота является большее преобщение русскоговорящих (и не только) пользователей к английскому языку.
На данный момент бот обладает указанным ниже функционалом, и способен работать на серверах не зависимо от их внутреннего устройства.
#
/traducere [par] [spar] - запускает мини-игру, целью которой является перевод предложенного слова с русского на английский язык или с английского на русский язык.
Ответ осуществляются по команде:
/reply [слово-перевод] [слово-перевод].
Возможные параметры (par, spar):
loop - зацикливает игрвой процесс до первой ошибки, или до ввода специальной команды отмены.
ru/en - устанавливает язык, с которого пользователь будет переводить слова.
#
/words - запускает мини-игру, целью которого является продолжение цепочки слов с последней буквы данного слова.
Ответ осуществляются по команде:
/reply [слово-продолжение].
#
/countries [par] - запускает мини-игру, целью которой является проверка знаний названий столиц соотвествующих стран на английском языке.
Ответ осуществляются по команде:
/reply [цифра-ответ].
Возможные параметры:
loop - зацикливает игровой процесс до первой ошибки, или до ввода специальной команды отмены.
#
/cancel - отменяет ожидание ответа от пользователя.
#
/timer [par] [spar] - запускает таймер, либо изменяет время на ответ пользователя.
Возможные параметры (par, spar):
I. 
[цифра] - число секунд, на которые будет установлен таймер (до 300).
II.
set - указатель на следующий параметр.
[цифра] - число секунд, дающихся на ответ пользователю при использовании некоторых команд.
!При вызове /timer без параметров, будет выведено текущее время на ответ пользователя.
#
/language [par]- устанавливает язык бота.
Возможные параметры:
ru/en - язык, на котором будут выводиться ответы бота пользователю.
!При вызове /language без параметров, будет выведен текущий язык бота.
#
/top [par] - выводит список лидеров сервера по очкам.
Возможные параметры (par):
[цифра] - количество выводимых строк списка лидеров.
#
/statistic - выводит личную статистику пользователя:
Количество очков;
Место в топе;
Комбо;
Максимальное комбо;
